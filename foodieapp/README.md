# Foodie

Foodie is a simple Node-based food delivery login app for a DevOps demo project.

## Demo users

- Jane: `jane` / `jane123`
- Ken: `ken` / `ken123`
- Fredd: `fredd` / `fredd123`

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Build for production

```bash
npm run build
```

The production files will be generated in `dist/`.
